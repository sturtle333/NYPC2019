pipe
======
- 원제: 수도관
- 성공
  - 풀이: 수도관 객체에 수도관이 이어질 수 있는지 최댓 최솟값을 저장, 비교하여 검사하는 메소드와 그게 아닐시 수도관을 이동시키는 메소드 두 개로 반복문을 돌림. 그리디 알고리즘.
  - 후기: 개인적으로 2일 차 문제부터 어렵다고 생각했다. 근데 다행히도 이건 쉬워서 편하게 푼 것 같다. 쉬어가는 문제같은 느낌.

======
- 원제: 신호등
- 성공
